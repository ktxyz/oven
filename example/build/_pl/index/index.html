oven

Kolejny statyczny generator stron z Markdownem

Dlaczego

Nie cierpię współczesnych stron internetowych. Tona JS, frameworki CSS i mnóstwo zbloatowanego bloatu. Dla mnie sieć ma być prosta

Featurki

  • Generowanie z plików .md przy użyciu szablonów Jinja2
  • Obsługa witryn wielojęzycznych z plikami .po
  • Wsparcie dla niestandardowych:
    • Filtrów Jinja
    • Rozszerzeń Markdown
    • Skryptów Pythona przed/po kompilacji
  • Brak bloatu, Markdown przekształca się w HTML

Użycie

Sprawdź plik README.md projektu, aby uzyskać więcej informacji na temat użytkowania i konfiguracji.

Korzystanie z oven’a jest bardzo proste.

Aby wygenerować stronę wystarczy uruchomić

~ oven build

w katalogu głównym witryny.

Tłumaczenia

Tłumaczenia są automatycznie ładowane z plików .po w skonfigurowanym katalogu.

Aby wygenerować/zaktualizować te pliki o nowe tłumaczenia wystarczy uruchomić

~ oven gather

w katalogu głównym witryny.
